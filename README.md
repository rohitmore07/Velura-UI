# Velura UI

**Velura UI** is a sleek, responsive and customizable UI library that simplifies the development of modern web applications. It provides a collection of pre-designed components and intuitive design patterns, enabling developers to build beautiful user interfaces with ease.

## :rocket: Live Preview : [check here](https://velura-ui.vercel.app/)

## ✨ Features

- Responsive and mobile-friendly design
- Easy-to-use pre-built components
- Fully customizable and extendable
- Optimized for performance and accessibility
- Compatible with modern web browsers
